# vuecesium 运行环境

> created by 20200916

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production and view the bundle analyzer report
npm run build --report
```
> 依赖安装成功之后，运行 访问 http://localhost:8080/#/
